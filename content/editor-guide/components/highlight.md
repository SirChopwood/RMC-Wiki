# Highlight

The `highlight` component allows you to visually emphasize specific words or phrases inside normal text.

It can be used directly inside paragraphs, lists, or other Markdown content.

---

## Usage

Basic syntax:

```md
This is an :highlight[important message].
```

Result:

This is an :highlight[important message].

---

## Usage in Lists

The component also works inside lists.

```md
- Normal list item
- An :highlight[important point]
```

Result:

* Normal list item
* An :highlight[important point]
